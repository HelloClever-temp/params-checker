## Installation
Add this line to your application's Gemfile:

```ruby
gem 'params_checker'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem 'params_checker', git: 'git@gitlab.com:rexylab/params-checker.git', branch: 'development'
```
